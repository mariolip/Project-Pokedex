As of right now, the Type Chart is still incomplete. While it isn't a difficult thing to put together, time simply hasn't been on our side.

For how the type chart works, it's simply supposed to be an HTML table, using images to represent the different types and how effective they

are against each other. There's really nothing more to it than a visual of what's strong and weak against what. Additionally, there would

also be a simple legend for the chart, detailing what each symbol means.


Note: Project Pokedex is NOT a comphresive website with every bit of Pokemon information. This website is intended to make getting into the series easier

in a fun and interactive way with a Pokedex in similar style that found in Pokemon Scarlet and Pokemon Violet.
